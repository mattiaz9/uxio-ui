Opinionated shadcn/ui extensions for layouts, inputs, overrides, and app-level layers, distributed
via the [shadcn registry](https://ui.shadcn.com/docs/registry).

## Component Categories

- **[Bundles](/docs/bundle)**: One-command installs for styles, overrides, layout, layers, or the
  full set
- **[Overrides](/docs/overrides/radix/alert)**: Enhancements to existing shadcn components (Base UI
  and Radix variants)
- **[Inputs](/docs/inputs/radix/input-currency)**: Specialized input fields (currency, datetime,
  duration, number, password, and more)
- **[Layout](/docs/layout/radix/split-content)**: Page and shell layout (e.g. split columns,
  Vercel-style sidebar drill-down)
- **[Layers](/docs/layers/radix/alerter)**: App-level UI layers (e.g. imperative **Alerter** and
  **Confirmer**)

## Installation

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Add components using the shadcn CLI.

```bash
npx shadcn@latest add @uxio/button
```

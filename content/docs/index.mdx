Fine-tuned shadcn/ui components plus something more... Distributed via the
[shadcn registry](https://ui.shadcn.com/docs/registry).

## Component Categories

- **[Bundles](/docs/bundle)**: One-command installs for styles, overrides, layout, layers, or the full set
- **[Overrides](/docs/overrides/radix/button)**: Enhancements of existing shadcn components (Base UI
  and Radix variants)
- **[Layout](/docs/layout/radix/split-content)**: Page and shell layout (e.g. split columns, Vercel-style sidebar drill-down)
- **[Layers](/docs/layers/radix/alerter)**: App-level UI layers (e.g. imperative **Alerter** and **Confirmer**)

## Installation

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Add components using the shadcn CLI. The correct variant (base or radix) is resolved automatically
from your `components.json` style.

```bash
npx shadcn@latest add @uxio/button
```

The Alert displays important messages to users. Install when your style is `base-nova`,
`base-vega`, etc.

## Install

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Then run:

```bash
npx shadcn@latest add @uxio/alert
```

The correct variant (base or radix) is resolved automatically from your `components.json` style.

## Extended Features

This override extends the original shadcn Alert with additional semantic color variants.

- [Semantic Colors](#semantic-colors)
- [Success Variant](#success-variant)
- [Info Variant](#info-variant)
- [Warning Variant](#warning-variant)

### Semantic Colors

The Alert depends on the `semantic-colors` registry style extension, which adds these CSS variables
automatically when you install the alert. The same values are used for both light and dark mode:

| Variable   | Value                    |
| ---------- | ------------------------ |
| `--success`| `oklch(0.62 0.36 145.52)`|
| `--info`   | `oklch(0.71 0.45 250)`   |
| `--warning`| `oklch(0.73 0.16 72.83)` |

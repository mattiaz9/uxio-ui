## Extended Features

- [Success](#success-variant), [Info](#info-variant), and [Warning](#warning-variant) color variants

## Install

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Then run:

```bash
npx shadcn@latest add @uxio/badge
```

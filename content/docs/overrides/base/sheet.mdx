## Extended Features

- `SheetContent` uses `bg-popover` instead of `bg-background`

## Install

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Then run:

```bash
npx shadcn@latest add @uxio/sheet
```

The correct variant (base or radix) is resolved automatically from your `components.json` style.

### Sheet Background

The sheet panel uses `bg-popover` instead of `bg-background` for a subtle visual distinction from the
page background.

The Button uses `@base-ui/react/button` as the primitive. Install when your style is `base-nova`, `base-vega`, etc.

## Install

```bash
npx shadcn@latest add @uxio/button
```

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

The correct variant (base or radix) is resolved automatically from your `components.json` style.

## Usage

```tsx
import { Button } from "@/components/ui/button"

<Button variant="outline">Button</Button>
```

## Variants

- **default**: Primary button style
- **outline**: Bordered outline style
- **secondary**: Secondary background
- **ghost**: Minimal hover style
- **destructive**: For dangerous actions
- **link**: Text link style

## Sizes

- **default**, **xs**, **sm**, **lg**
- **icon**, **icon-xs**, **icon-sm**, **icon-lg**

### As Link

Base UI Button does not support `asChild`. Use `buttonVariants` with an `<a>` tag for link styling:

```tsx
import { buttonVariants } from "@/components/ui/button"

<a href="#" className={buttonVariants({ variant: "secondary", size: "sm" })}>
  Login
</a>
```

## Extended Features

- Same as the original shadcn input-group component

## Install

This component requires `button`, `input`, and `textarea`. Install them from the default shadcn
registry first if you haven't already:

```bash
npx shadcn@latest add button input textarea
```

Then configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

And run:

```bash
npx shadcn@latest add @uxio/input-group
```

The correct variant (base or radix) is resolved automatically from your `components.json` style.

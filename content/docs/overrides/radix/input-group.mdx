## Extended Features

- [Size variants](#size-variants) — `xs`, `sm`, `default`, and `lg` on `InputGroup`, aligned with the
  [Input](/docs/overrides/radix/input) override. `InputGroupInput` and `InputGroupButton` inherit the
  group size when you omit their own `size` prop.

## Install

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Then run:

```bash
npx shadcn@latest add @uxio/input-group
```

The correct variant (base or radix) is resolved automatically from your `components.json` style.

## Usage

Set `size` on `InputGroup` so the shell height matches the inner `Input` (and optional addons). Pass
`size` explicitly on `InputGroupInput` or `InputGroupButton` only when you need a different size
than the group.

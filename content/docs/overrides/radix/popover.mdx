## Extended Features

- `PopoverContent` uses `bg-popover` instead of `bg-background`

## Install

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Then run:

```bash
npx shadcn@latest add @uxio/popover
```

The correct variant (base or radix) is resolved automatically from your `components.json` style.

### Popover Background

The popover content uses `bg-popover` instead of `bg-background` for a subtle visual distinction
from the page background.

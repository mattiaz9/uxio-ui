## Extended Features

- Support for recharts@3.8.0
- Tooltip uses `bg-popover` instead of `bg-background`
- Support for stricter lint rules

## Install

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Then run:

```bash
npx shadcn@latest add @uxio/chart
```

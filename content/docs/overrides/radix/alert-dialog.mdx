The Alert Dialog uses `radix-ui` for the primitive. Install when your style is `radix-nova`,
`radix-vega`, etc.

## Extended Features

This override extends the original shadcn Alert Dialog with a content background change.

- [Popover Background](#popover-background)

## Install

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Then run:

```bash
npx shadcn@latest add @uxio/alert-dialog
```

The correct variant (base or radix) is resolved automatically from your `components.json` style.

### Popover Background

The dialog content uses `bg-popover` instead of `bg-background` for a subtle visual distinction
from the page background.

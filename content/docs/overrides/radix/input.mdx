## Extended Features

- [Size variants](#size-variants) — `xs`, `sm`, `default`, and `lg` with the same dimensions and
  spacing as the Button component

## Install

Configure your registry in `components.json`:

```json
{
  "registries": {
    "@uxio": "https://ui.uxio.dev/r/styles/{style}/{name}.json"
  }
}
```

Then run:

```bash
npx shadcn@latest add @uxio/input
```

The correct variant (base or radix) is resolved automatically from your `components.json` style.
